# Arquitetura

Documentos de arquitetura do **Caminho da Erudição** — uma aplicação de
questões de matemática do Ensino Fundamental — Anos Finais, alinhada à BNCC,
com sistema multiusuário, ranking e recompensas.

## Documentos desta seção

- **[Visão Geral](visao-geral.md)** — componentes lógicos, responsabilidades, fluxos principais e princípios de design da aplicação.
- **[Modelo de Dados](modelo-de-dados.md)** — entidades, relações, índices, regras de integridade e auditoria.
- **[Tecnologias](tecnologias.md)** — critérios de escolha de stack e alternativas em consideração.
