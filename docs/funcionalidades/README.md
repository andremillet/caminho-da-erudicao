# Funcionalidades

Detalhamento das funcionalidades da aplicação **Caminho da Erudição**.

## Documentos desta seção

- **[Usuários](usuarios.md)** — cadastro, autenticação, perfil, configurações, segurança e conformidade LGPD.
- **[Banco de Questões](questoes.md)** — estrutura da questão, classificação BNCC, dificuldade, modificadores de pontuação, ciclo de vida.
- **[Ranking](ranking.md)** — múltiplos rankings (por escopo e período), regras de pontuação e desempate, anti-trapaça.
- **[Recompensas](recompensas.md)** — sistema híbrido de badges (conquistas) e moedas (economia virtual), com exemplos de uso.
